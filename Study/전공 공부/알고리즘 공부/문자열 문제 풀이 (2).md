    - 10798
        
        - 문제 링크: [https://www.acmicpc.net/problem/10798](https://www.acmicpc.net/problem/10798)
            
        - 난이도: 브론즈 1
            
        - 내 풀이
            
            ```jsx
            import sys
            input = sys.stdin.readline
            
            words = ["" for _ in range(5)]
            
            max_length = -1
            
            for i in range(5):
                words[i] = input().rstrip()
                if max_length < len(words[i]):
                    max_length = len(words[i])
            
            for j in range(max_length):
                for i in range(5):
                    if j >= len(words[i]):
                        continue
                    print(words[i][j], end="")
            ```

    - 1157
        
        - 문제 링크: [https://www.acmicpc.net/problem/1157](https://www.acmicpc.net/problem/1157)
            
        - 난이도: 브론즈 1
            
        - 내 풀이
            
            ```jsx
            import sys
            input = sys.stdin.readline
            
            word = input().rstrip()
            word = word.lower()
            frequency = [0] * 26
            
            for i in range(len(word)):
                frequency[ord(word[i]) - ord('a')] += 1
            
            m = max(frequency)
            
            if frequency.count(m) >= 2:
                print("?")
            else:
                print(chr(frequency.index(m) + ord('a')).upper())
            
            ```
            
        - gpt 풀이 참고
