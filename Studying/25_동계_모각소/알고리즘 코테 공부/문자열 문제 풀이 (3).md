    - 1764
        
        - 문제 링크: [https://www.acmicpc.net/problem/1764](https://www.acmicpc.net/problem/1764)
            
        - 난이도: 실버 4
            
        - 내 풀이
            
            ```jsx
            import sys
            input = sys.stdin.readline
            
            N, M = map(int, input().split())
            
            not_heard = set()
            not_heard_seen = set()
            cnt = 0
            
            for _ in range(N):
                not_heard.add(input().rstrip())
                
            for _ in range(M):
                not_seen = input().rstrip()
                if not_seen in not_heard:
                    not_heard_seen.add(not_seen)
                    cnt += 1
            
            ans_list = list(not_heard_seen)
            
            ans_list.sort()
            
            print(cnt)
            for i in range(len(ans_list)):
                print(ans_list[i])
            
            ```
            
        - 모범 답안
            
            ```jsx
            import sys
            input = sys.stdin.readline
            
            N, M = map(int, input().split())
            
            not_heard = {input().rstrip() for _ in range(N)}
            not_seen = {input().rstrip() for _ in range(M)}
            
            ans = sorted(not_heard & not_seen)
            
            print(len(ans))
            print('\\n'.join(ans))
            ```
            
    - 20920
        
        - 문제 링크: [https://www.acmicpc.net/problem/20920](https://www.acmicpc.net/problem/20920)
            
        - 난이도: 실버 3
            
        - 내 풀이
            
            ```jsx
            import sys
            
            input = sys.stdin.readline
            
            N, M = map(int, input().split())
            
            words = {}
            
            for _ in range(N):
                word = input().rstrip()
                if len(word) < M:
                    continue
                words[word] = words.get(word, 0) + 1
            
            sorted_words = sorted(words.items(), key=lambda x: (-x[1], -len(x[0]), x[0]))
            
            for w in sorted_words:
                print(w[0])
            ```
            
        - 모범 답안
            
            ```jsx
            import sys
            from collections import Counter
            
            input = sys.stdin.readline
            
            N, M = map(int, input().split())
            
            words = []
            
            for _ in range(N):
                w = input().rstrip()
                if len(w) >= M:
                    words.append(w)
            
            counter = Counter(words)
            
            result = sorted(
                counter.items(),
                key=lambda x: (-x[1], -len(x[0]), x[0])
            )
            
            for word, _ in result:
                print(word)
            ```
