    - 9046
        
        - 문제 링크: [https://www.acmicpc.net/problem/9046](https://www.acmicpc.net/problem/9046)
            
        - 난이도: 브론즈 2
            
        - 내 풀이
            
            ```jsx
            import sys
            
            input = sys.stdin.readline
            
            n = int(input())
            
            for _ in range(n):
                frequency = [0] * 26
                password = input().rstrip()
                
                for p in password:
                    if p == ' ':
                        continue
                    
                    frequency[ord(p) - ord('a')] += 1
                    
                max_freq = max(frequency)
                
                if frequency.count(max_freq) > 1:
                    print("?")
                else:
                    print(chr(frequency.index(max_freq) + ord('a')))
            ```
