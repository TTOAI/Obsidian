    - 10798
        
        - 문제 링크: [https://www.acmicpc.net/problem/10798](https://www.acmicpc.net/problem/10798)
            
        - 난이도: 브론즈 1
            
        - 내 풀이
            
            ```jsx
            import sys
            input = sys.stdin.readline
            
            words = ["" for _ in range(5)]
            
            max_length = -1
            
            for i in range(5):
                words[i] = input().rstrip()
                if max_length < len(words[i]):
                    max_length = len(words[i])
            
            for j in range(max_length):
                for i in range(5):
                    if j >= len(words[i]):
                        continue
                    print(words[i][j], end="")
            ```

    - 1157
        
        - 문제 링크: [https://www.acmicpc.net/problem/1157](https://www.acmicpc.net/problem/1157)
            
        - 난이도: 브론즈 1
            
        - 내 풀이
            
            ```jsx
            import sys
            input = sys.stdin.readline
            
            word = input().rstrip()
            word = word.lower()
            frequency = [0] * 26
            
            for i in range(len(word)):
                frequency[ord(word[i]) - ord('a')] += 1
            
            m = max(frequency)
            
            if frequency.count(m) >= 2:
                print("?")
            else:
                print(chr(frequency.index(m) + ord('a')).upper())
            
            ```

    - 4659
        - 문제 링크: [https://www.acmicpc.net/problem/4659](https://www.acmicpc.net/problem/4659)
            
        - 난이도: 실버 5
            
        - 내 풀이
            
            ```jsx
            import sys
            input = sys.stdin.readline
            
            while True:
                password = input().rstrip()
                if password == "end":
                    break
                
                acceptable = False
                consonant = 0
                vowel = 0
                prev = 0
                for p in password:
                    if p == 'a' or p == 'e' or p == 'i' or p == 'o' or p == 'u':
                        acceptable = True
                        vowel = 0
                        consonant += 1
                    else:
                        consonant = 0
                        vowel += 1
                    if consonant >= 3 or vowel >= 3:
                        acceptable = False
                        break
                    if prev == p:
                        if (prev == 'e' and p == 'e') or (prev == 'o' and p == 'o'):
                            acceptable = True
                        else:
                            acceptable = False
                            break
                    prev = p
                
                if acceptable == False:
                    print("<", password, ">", " is not acceptable.", sep = "")
                else:
                    print("<", password, ">", " is acceptable.", sep = "")
                    
            ```
            
        - gpt 풀이 참고
            
            ```jsx
            import sys
            input = sys.stdin.readline
            
            vowels = {'a', 'e', 'i', 'o', 'u'}
            
            while True:
                password = input().strip()
                if password == "end":
                    break
            
                has_vowel = False
                vowel_cnt = 0
                consonant_cnt = 0
                acceptable = True
            
                for i in range(len(password)):
                    c = password[i]
            
                    # 1️⃣ 모음 여부
                    if c in vowels:
                        has_vowel = True
                        vowel_cnt += 1
                        consonant_cnt = 0
                    else:
                        consonant_cnt += 1
                        vowel_cnt = 0
            
                    # 2️⃣ 모음 3개 or 자음 3개 연속
                    if vowel_cnt == 3 or consonant_cnt == 3:
                        acceptable = False
                        break
            
                    # 3️⃣ 같은 글자 연속 (ee, oo 허용)
                    if i > 0 and password[i] == password[i-1]:
                        if c not in {'e', 'o'}:
                            acceptable = False
                            break
            
                # 4️⃣ 모음 포함 여부
                if not has_vowel:
                    acceptable = False
            
                if acceptable:
                    print(f"<{password}> is acceptable.")
                else:
                    print(f"<{password}> is not acceptable.")
            ```
