
- 요점 정리
	- memory 안의 값은 바로 접근 불가, register를 통해 접근 가능
	- MIPS는 32-bit(4 byte) register 32개 가짐
	- MIPS에서 word는 32-bit(4 byte) (architecture마다 크기 다름)
	- sw는 왼쪽에서 오른쪽 방향으로 저장
	- memory 주소 하나의 크기는 1byte(1byte에 2진수 8자리, 16진수 2자리)
	- immediate operands
		- addi (subi 없음, negative constant 이용)
		- 무조건 register 하나에 constant 하나, constant 2개 X
		- 순서: register -> constatnt
		- $zero를 통해 register간 값 복사, register에 상수값 저장 가능
