# 1과목 - 데이터의 이해
### 데이터의 유형
- 정형 데이터 - RDB, 엑셀 스프레드시트, CSV
- 반정형 데이터: 메타데이터 포함 - 로그, HTML, XML, JSON
- 비정형 데이터 - SNS, 유튜브, 음원
### 암묵지, 형식지
- 암묵지: 개인에게 습득, 겉으로 드러나지 않음
- 형식지: 문서, 매뉴얼 등의 형상화된 지식
1. 공통화: 암묵지 지식을 다른 사람에게 알려줌
2. 표출화: 암묵지 지식을 매뉴얼이나 문서로 전환
3. 연결화: 교재, 매뉴얼에 새로운 지식 추가
4. 내면화: 만들어진 교재, 매뉴얼에서 다른 사람의 암묵지를 터득
### DIKW 피라미드
1. 데이터(Data): 있는 그대로
2. 정보(Information): Data를 통해 패턴 인식
3. 지식(Knowledge): 패턴을 통해 예측
4. 지혜(Wisdom): 창의적인 산물
### 데이터 단위
- K < M < G < T < P < E < Z < Y
- 2^10 < ... < 2^80 
### 데이터베이스의 특징
- 공용 데이터
- 통합된 데이터: 데이터 중복 X
- 저장된 데이터: 저장매체에 저장
- 변화되는 데이터: 현재의 정확한 데이터는 유지(무결성)
### 기업 활용 데이터베이스
- OLTP: OnLine Transaction Processing, 데이터를 수시로 갱신
- OLAP: OnLine Analytical Processing, 다차원 데이터를 대화식으로 분석
- CRM: Customer Relationship Management, 고객과 관련 자료 분석, 마케팅 활용
- SCM: Supply Chain Management, 공급망 연결 최적화
- ERP: Enterprise Resource Planning, 기업 경영 자원을 효율화
- BI: Business Intelligence, 기업 보유 데이터 정리, 분석하여 리포트 중심 도구
- BA: Business Analytics, 통계 기반 비즈니스 통찰력
- Block Chain: 네트워크 참여한 모든 사용자가 정보를 분산, 저장
- KMS: Knowledge Management System, 기업의 모든 지식을 포함
### Data Ware House
- 주제지향성
- 데이터 통합
- 시계열성
- 비휘발성: 읽기 전용, 수시로 변하지 않음
### ETL
- 데이터를 DW와 DM 등의 DB 시스템에 적재하는 데 사용
### Data Lake
- 비정형 데이터를 저장하며 하둡과 연계하여 처리
- 하둡: 병렬처리 오픈소스 프레임워크
### 빅데이터의 3V
- Volume, Variety, Velocity
### 빅데이터가 만들어내는 변화
- 표본조사 -> 전수조사
- 사전처리 -> 사후처리
- 질 -> 양
- 인과관계 -> 상관관계
- "전후양상" 외우기

- "인자기" 외우기
### 빅데이터 주요 분석기법

# 3과목 - 데이터 분석
### 데이터 마트
- 특정 목적을 위해 사용하는 데이터 웨어하우스의 한 분야
### EDA
- EDA의 4가지 주제: 저항성의 강조, 잔차 계산, 자료변수의 재표현, 그래프를 통한 현시성
### 이상값
- 항상 제거 X
- ESD: 표준편차의 3배 넘어가는 데이터는 이상값
- 사분위수: Q1 - 1.5IQR보다 작거나 Q3 + 1.5IQR보다 크면 이상값
### 표본 추출 방법
- 계통 추출법: 번호 부여, 일정 간격으로 추출
- 집략 추출법: 군집 내 이질적
- 층화 추출법: 군집 간 이질적
### 자료의 척도 구분
- 질적 척도: 명목척도, 순서척도
- 양적 척도: 등간척도, 비율척도
- 등간척도: 구간 사이의 간격이 의미있음, 덧셈 뺄셈 가능 (ex. 온도)
- 비율척도: 절대적 기준 0이 존재, 사칙연산 가능 (무게, 나이)
### 공분산
- 두 확률변수의 상관정도
- = 0: 상관 없음
- > 0: 양의 상관관계
- < 0: 음의 상관관계
- 최소, 최대값이 없어 강약 판단 불가
### 상관계수
- 상관정도를 -1 ~ 1값으로 표현
- = 1: 정비례 관계
- = -1: 반비례 관계
### 첨도
- = 0 or 3: 정규 분포 형태
### 왜도
- 왜도 < 0: 최빈값 > 중앙값 > 평균값
- 왜도 > 0: 최빈값 < 중앙값 < 평균값
### 연속 확률분포
- 정규분포: 가우스분포, Z 검정
- t분포: 두 집단의 평균치 차이의 비교 검정 시 사용 (T 검정), 데이터 개수가 30개 이상이면 정규성 검정 불필요
- 카이제곱분포: 두 집단의 동질성 검정, 혹은 단일 집단 모분산에 대한 검정 (카이제곱 검정)
- F분포: 두 집단의 동일성 검정 시 사용 (F 검정)
### 가설검정
- 귀무가설: 일반적으로 생각하는 가설
- 대립가설: 귀무가설을 기각하는 가설, 증명하고자 하는 가설 (차이가 있다/크다/작다)
- 유의수준: 1종 오류를 범할 확률의 허용 한계 (일반적으로 0.05)
- 유의확률: 귀무가설을 지지하는 정도를 나타내는 확률
- 1종 오류: 귀무가설 사실, 검정결과 거짓
- 2종 오류: 귀무가설 거짓, 검정결과 사실
### 비모수검정
- 모집단에 대한 아무런 정보가 없을 때
- 관측 자료가 특정 분포를 따른다고 가정이 불가할 때
- 부호검정, 순위합검정, 만-휘트니 U검정, 크러스컬-윌리스 검정
### 회귀분석
- 회귀계수(가중치) 추정방법: 최소제곱법
- 회귀모형 평가: R-squared
### 회귀분석의 가정
- 선형성: 종속변수와 독립변수는 선형관계
- 등분산성: 잔차의 분산이 고르게 분포
- 정상성(정규성): 잔차가 정규분포의 특성을 지님
	- Q-Q plot, 샤피로 윌크 검정, 히스토그램, 왜도와 첨도 활용
- 독립성: 독립변수들 간 상관관계가 없음
### 회귀분석의 분산분석표
- 회귀
	- 제곱합: SSR
	- 자유도: p
	- 제곱평균: MSR = SSR / p
- 잔차
	- 제곱합: SSE
	- 자유도: n - p - 1
	- 제곱평균: MSE = SSE / n - p - 1
- 결정계수 = SSR / SST = 1 - (SSE / SST)
- Adjusted R-square = 1 - (n - 1) * MSE / SST
	- 다중회귀에서 사용
### 회귀 모형의 검정
1. 독립변수와 종속변수 설정
2. 회귀계수 값의 추정
3. 모형이 통계적으로 유의미한가
	- 모형에 대한 F 통계랑, p-value
	- 귀무가설: 모든 회귀계수는 0이다
4. 회귀계수들이 유의미한가
	- 회귀계수들의 t 통계량, p-value
	- 각 회귀계수에 대한 귀무가설: 회귀계수는 0이다
5. 모두 기각하면 해당 모델 활용
6. 모형이 설명력을 갖는가: 결정계수(R-square) 값
### 최적의 회귀방정식 탐색 방법
- 전진선택법: 변수 하나씩 추가
- 후진제거법: 변수 하나씩 제거
- 단계별 선택법: 벌점 고려
	1. AIC(아카이케 정보 기준)
		- 편향과 분산이 최적화되는 지점 탐색
		- 자료가 많을수록 부정확
	2. BIC(베이즈 정보 기준)
		- AIC 보완, AIC보다 큰 패널티를 가짐
		- 변수가 적은 모델에 적합
### 상관분석
- 피어슨 상관분석: 양적 척도, 연속형 변수, 선형관계 크기 측정
- 스피어만 상관분석: 서열 척도, 순서형 변수, 선형/비선형적 관계 나타냄
- 다중공선성: 변수 간 상관관계 클 때 모델을 불안정하게 만듦
### 다차원 척도법
- 데이터 축소 목적
- Stress 값이 0에 가까울수록 좋음
- 계량적 MDS: 양적척도 활용
- 비계량적 MDS: 순서척도 활용
### 주성분 분석(PCA)
- 상관성 높은 변수들의 선형 결합으로 차원 축소
- 자료의 분산이 가장 큰 축이 첫 번째 주성분
- 스크리플롯
	- 주성분 개수 선택에 도움
	- 수평을 이루기 바로 전 단계의 개수로 선택
- 바이플롯
	- PC와 평행할수록 해당 PC에 큰 영향
### 시계열 분석
- 시간의 흐름에 따라 관찰된 자료의 특성을 파악하여 미래를 예측 (ex. 주가 데이터, 기온 데이터)
- 정상성: 시계열 예측을 위해서는 모든 시점에 일정한 평균과 분산을 가지는 정상성을 만족해야 함
- 차분: 현 시점의 자료를 이전 값으로 빼는 방법
-  백색 잡음: 시계열 모형의 오차항, 원인 모름
	- 평균이 0 -> 가우시안 백색잡음
### 시계열 모형
- 자기회귀(AR) 모형
	- 자기자신의 과거 값이 미래 결정
	- 부분자기상관함수(PACF) 활용, p+1 시점 이후 급격 감소하면 AR(p) 모형 선정
- 이동평균(MA) 모형
	- 이전 백색잡음들의 선형결합으로 표현되는 모형
	- 자기상관함수(ACF)를 활용, q+1 시점 이후 급격 감소하면 MA(q) 모형 선정
- 자기회귀누적이동평균(ARIMA) 모형
	- AR 모형과 MA 모형의 결합

### 연관분석
- 항목들 간의 조건-결과로 이루어지는 패턴을 발견하는 기법 (장바구니 분석)

- Apriori 알고리즘 활용
- 순차패턴: 연관분석에 시간 개념 추가
### 연관분석의 지표
- 지지도: 두 품목이 동시에 포함된 거래 비율
- 신뢰도: A 품목이 거래될 때 B품목도 거래될 확률 (조건부 확률)
- 향상도: A 품목과 B 품목의 상관성
	- > 1: 양의 상관관계
	- = 1: 상관 없음
	- < 1: 음의 상관관계
