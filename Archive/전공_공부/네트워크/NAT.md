
## NAT(Network Address Translation)

- 사설망과 공인망 사이에서, IP 주소를 바꿔주는 라우터 기능
- 보통 가정/회사의 공유기가 이 역할을 함
- IPv4 주소 부족 문제 해결
- 외부에서 내부 IP를 직접 알 수 없으므로 보안 기능
- 종류:
	- Static NAT
		- 사설 IP와 공인 IP를 1:1로 고정 매핑
	- Dynamic NAT
		- 가용한 공인 IP 풀(Pool)이 존재
	- PAT/NAPT(Network Address Port Translation)
		- 하나의 공인 IP를 여러 내부 장비가 포트 번호를 이용해 공유

## 포트 포워딩(Port Forwarding)

- NAT(공유기/라우터)가 외부에서 들어오는 특정 포트 요청을 내부 사설 IP의 특정 장치/포트로 전달해주는 기능
- 예시:
	1. 공유기 NAT에 규칙 추가:
		- `203.0.113.10:8080 → 192.168.0.5:80`
	2. 누군가 인터넷에서 `http://203.0.113.10:8080`으로 접속
	3. 공유기 NAT가 이 요청을 보고 내부 `192.168.0.5:80`으로 전달
