- 자연수: 헤아릴 수 있는 무한
- 실수: 헤아릴 수 없는 무한

- 자연수, 짝수 -> 원소 개수 같음
- 실수, 유리수 -> 실수의 개수 > 유리수의 개수

- 멱집합
	- 실수보다 더 많은 무한대 존재
	- 주어진 집합의 모든 부분집합들을 원소로 갖는 집합
	- 멱집합으로 얻어진 무한은 원래 무한보다 크기가 큼

- 무한대의 크기, 종류가 무한히 많음
	- 무한집합의 개수를 타나낼 수 있는 기호가 필요함
	- 가산집합(= 자연수 집합의 크기) = 알레프 0
	- 가산집합의 멱집합의 크기(= 실수의 개수) = 알레프 1
	- 실수의 멱집합의 크기 = 알레프 2

- 알레프0보다 크고 알레프1보다 작은 집합이 존재할까?
- 연속체 가설
	- 힐베르트 문제의 첫번째 문제
	- 모든 알파에 대하여 알레프 알파보다 크고 알레프 알파+1보다 작은 크기를 갖는 집합은 없다
	- 맞지도 틀리지도 않다
	- 쿠르트 괴델
		- ZFC 공리계에서는 연속체 가설의 반증이 불가능하다
		- 불완전성의 정리로 증명함
	- 폴 코헨
		- 연속체 가설이 틀렸다고 가정해도 모순이 발생하지 않는다
