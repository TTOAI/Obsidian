- 건달: 수학에서 항상 거짓말만 하는 사람을 부르는 표현
- 기사: 수학에서 항상 참인 말만 하는 사람을 부르는 표현
- 기사와 건달 문제

- 기사와 건달 문제1
	- ![[Pasted image 20250920184453.png]]
	- 성현이 기사라면 참인 말만 함
		- 건달이라고 했으니 거짓
		- 성현은 건달
	- 아현이 건달이라면 성현은 참을 말한 것
		- 성현의 말이 거짓이려면 아현은 기사

- 논리학의 모순: 두 명제가 동시에 참이거나 동시에 거짓일 수 없는 경우
	- 부정 관계의 두 명제를 동시에 이야기할 때
	- ex) 기사이다 + 건달이다

- 한비자에서 유래된 모순
	- 동시에 참일 수는 없지만 동시에 거짓일 수는 있음
	- 서로 부정 관계가 아님
	- 모순이 아님

- 역설: 이치에 맞지 않는 표현
	- 말이 안되는 문장
	- 모순도 역설에 포함됨
	- ex) 나는 건달이야

- 조건 명제의 부정
	- ![[Pasted image 20250920184138.png]]
	- '내가 기사면 전재산을 주겠다'의 부정 -> 내가 기사이고 전재산을 주지 않겠다
	- 성현이 건달이면 기사여야 함 -> 모순
	- 따라서 성현은 기사여야 함

- 삼단 논법
	- 아리스토텔레스의 논리학 중 하나
	- 대전제, 소전제에 따라 결론을 내리는 논리의 기본 틀
		- p -> q
		- q -> r
		- p -> r
