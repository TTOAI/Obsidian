
- 분리의 법칙: 대립유전자가 생식세포가 형성될 때 분리되는 것
- 동형접합자: PP
- 이형접합자: Pp
- 단성잡종: 한 가지 특성에 의한 이형접합자
- 독립의 법칙: 다른 특성들은 서로에게 영향을 미치지 않음
- 유전자 좌위: 염색체에서 특정 유전자가 위치한 자리
	- 유전자의 표현형에 대한 단백질을 암호화하는 특정 DNA 서열
- 불완전우성
	- 이형접합자에서 열성 대립유전자의 효과가 어느 정도 나타남
	- 낫모양적혈구빈혈증
- 공동우성
	- 이형접합자에서 대립유전자가 같은 수준으로 검출됨
	- MN 혈액형
- 다인자유전은 주위 환경에 의해 흔히 변형될 수 있음
- 다면발현: 단일 유전자가 여러 종류의 유전적 특성에 영향을 줌
	- 낫모양적혈구빈혈증
- 연관 유전자: 동일한 염색체에 위치한 유전자
- 두 유전자 사이의 거리가 가까울수록
	- 함께 유전될 확률, 부모형이 나타날 확률이 더 높음
	- 재조합형이 생산될 확률이 더 낮음
- SRY 유전자: 배 발생의 초기(7~8주)에 XY배에서 활성화되어 발생을 남성으로 전환
- 유전자 정량 메커니즘
	- 암컷의 체세포 대부분에서 두 X 염색체 중 하나를 불활성화시킴
	- 불활성화되어 응축된 X 염색체는 바소체로 관찰됨
		- 바소체: 염색질의 밀집된 덩어리
	- 불활성화는 배 발생동안 일어나고 각 세포마다 무작위로 선택됨
		- 비가역적이므로 해당 세포의 모든 후손은 같은 X 염색체가 불활성화됨

# 염색체의 변형
- 결실
	- 5번 염색체 결실: 고양이울음소리증후군
- 중복
	- 포유동물의, 상어와 다른 종류의 헤모글로빈 유전자
	- 감수분열 재조합 과정에서 교차의 불균등으로 나타나기도 함
- 전좌
	- 8번, 14번 염색체 상호전좌: 버킷림프종
	- 세포분열 조절 유전자 포함
	  -> 전좌된 유전자의 과잉 발현
	  -> 세포분열의 무절제한 발생
	  -> 암 발생
- 역위

# 감수분열에서 비분리현상
- 이수체: 일부 염색체가 사라지거나 여분의 염색체를 더 가진 개체
- 정배수체: 염색체의 정상적인 세트를 더 가진 개체
- 다운증후군: 21번 염색체 비분리현상
- 다배수체: 방추제가 정상적인 활동을 못하여 발생
	- 식물의 경우 다배수체가 성장과 생식에서 더 성공적

- 유전상담
	- 분만전 진단
	- 양수 검사
	- 융모막돌기 채취법
	- 유전학적 선별
