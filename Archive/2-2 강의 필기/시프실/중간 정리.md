# Program Block
- label의 실제 address 구하기
	- 해당 block의 실제 시작 address + block 안에서 label의 address
- block의 실제 시작 address 구하기
	- 이전 block들의 length의 합
- block의 length 구하기
	- block의 마지막 instruction의 byte 수 고려
	- block의 마지막 byte address + 1 (LOC이 0에서부터 시작했으므로)
- label의 실제 address
  = block 안에서 label의 address + 이전 block들의 length의 합
- directive는 block의 length에 포함 X
	- ex) EQU
- object file에서 주의점
	- directive는 object code는 없지만 address는 할당됨
	- 각 text record의 시작 address 계산할 때 고려해야함
