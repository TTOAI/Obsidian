- 실수: IEEE 754 부동소수점 변환에 따라 double precision으로 표현 (1비트, 11비트, 52비트)
- #: 실수는 소수점 이하 무시
- LDCH, RD: A register의 rightmost byte에 1byte 저장
- LDA: A register에 최대 3btye 저장
	- \\#으로 저장 시
		- 최대 3btye 크기 정수 저장 가능, 초과 시 오류
		- 실수로 저장 시 소수점 이하 무시
	- label로 저장 시
		- 정수, 실수 상관 없이 시작 메모리 주소부터 3btye만큼 저장
- LDF: F register에 최대 6btye 저장
	- \\#으로 저장 시
		- 최대 얼마까지 저장 가능한지 모름
		- 실수로 저장 시 소수점 이하 무시
	- label로 저장 시
		- 정수, 실수 상관 없이 시작 메모리 주소부터 6btye만큼 저장
- WORD: 최대 3btye 크기 정수 저장 가능, 초과 시 오류
- FLOT: 최대 6byte 크기 실수 저장 가능, 초과 시 초과 비트는 무시
- STA: 시작 메모리 주소부터 3btye 저장
- STF: 시작 메모리 주소부터 6byte 저장
- RESW: 시작 메모리 주소부터 개수 * 3btye만큼 예약
- RESF: 시작 메모리 주소부터 개수 * 6btye만큼 예약
