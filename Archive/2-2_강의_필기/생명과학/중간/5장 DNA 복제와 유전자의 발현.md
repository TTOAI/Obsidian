# tRNA
- tRNA (운반 RNA)
	- 아미노산을 리보솜으로 운반하는 RNA
	- 클로버잎 구조
- 안티코돈
	- tRNA 이중나선 부위 중 하나의 끝에 있는 부분
	- mRNA의 코돈과 염기쌍을 이루는 3개의 뉴클레오타이드 부분
	- 안티코돈의 반대쪽에는 안티코돈에 상응하는 아미노산에 결합된, 이중나선 부위가 있음
		- ex) 안티코돈과 코돈은 DNA 가닥에서처럼 역평행적 방향으로 쌍을 이룸
- 동요가설
	- 61개의 의미 있는 코돈의 완전한 세트는 안티코돈의 특이적인 염기쌍 형성 특징 때문에 61개보다 적은 수의 tRNA들에 의해 읽힘
		- 안티코돈은 코돈의 세 번째 뉴클레오타이드와의 염기 짝짓기에서는 유연성을 가짐
		- ex) 페닐알라닌을 운반하는 tRNA는 UUU와 UUC 코돈을 읽을 수 있음
- 아미노아실화: 아미노산을 대응되는 tRNA에 부착
	- 아미노아실 tRNA 합성효소
		- 아미노아실화 촉매
		- 20개 아미노산 각각에 하나의 합성효소

# 리보솜
- 자동화된 단백질 조립 기구로 작동
- 아미노산을 순서대로 연결함
- rRNA(리보솜 RNA)와 리보솜 단백질들의 복합체
- 번역 과정에서 mRNA는 리보솜 속 틈새를 통과해 이동
- tRNA, mRNA가 상호작용하는 결합자리 가짐

- A(아미노아실) 자리: 펩타이드 사슬에 더해질 다음 아미노산을 운반하는 아미노아실 tRNA가 mRNA에 결합하는 곳
- P(펩티딜결합부위) 자리: 중합되는 폴리펩타이드 사슬을 운반하는 tRNA가 결합하는 곳
- E(방출 자리) 자리: 리보솜에서 방출되기 전에 tRNA가 결합되는 자리

# 번역
- 번역의 주요 세 단계: 개시, 연장, 종결
- 번역 개시
	- 리보솜 큰, 작은 단위체가 mRNA와 결합
	- 첫 번째 아미노아실 tRNA가 AUG 개시코돈에 결합
	- 개시에 사용되는 아미노아실 tRNA는 메싸이오닌 운반
	- 개시가 끝날 때 Met-tRNA는 P 자리에 있음
	- mRNA 상의 뉴클레오타이드를 한 번에 3개씩 읽음
- 번역 연장
	- 개시 tRNA가 P 자리에 결합하고 A 자리는 비어 있는 채로 시작
	- 적절한 안티코돈을 갖는 아미노아실 tRNA가 A 자리 코돈에 결합
	- P 자리의 아미노산이 A 자리에 있는 아미노산으로 이동하여 펩타이드 결합을 형성함
	- 리보솜은 GTP 가수분해로부터의 에너지를 사용하여 mRNA를 따라 다음 코돈으로 이동함
	- 펩티딜-tRNA가 P자리에 있고 A 자리가 비어있음
		- 펩티딜-tRNA:여러 개의 아미노산이 결합된 폴리펩타이드 사슬을 운반하는 tRNA
	- E 자리로 이동한 빈 tRNA는 리보솜으로부터 방출됨
	- 리보솜의 A 자리가 종결코돈 중의 하나에 도달함
- 번역 종결
	- 방출인자(종결인자)가 아미노아실 tRNA 대신 A 자리에 결합
	- 방출과 해체

# 염기쌍 대체 돌연변이
- 과오 돌연변이
	- 염기쌍 변화가 정상적인 것과 다른 아미노산 서열을 만듦
	- 낫모양적혈구빈혈증: 헤모글로빈의 폴리펩타이드 종류 중 하나의 유전자 내 과오 돌연변이에 의해 발생함
	- 많은 다른 인간 유전병: 동형 접합성 과오 돌연변이
- 정지 돌연변이
	- 염기쌍 변화가 전사 코돈을 정지 코돈으로 바꿈
- 침묵 돌연변이
	- 염기쌍 변화로 변화된 코돈이 정상적인 것과 동일한 아미노산을 지정함
- 번역틀이동 돌연변이
	- 염기 하나가 삭제 또는 삽입됨

- 원핵생물은 주변 환경에 맞춰 효소 생산을 중단하거나 재개하는 등 유전자 발현을 조절함
- 대장균에서 젖당 대사를 위한 효소를 암호하는 3개의 유전자들은 젖당의 유무에 따라 발현되거나 발현되지 않음
- 오페론: 원핵생물의 유전자들과 이들의 조절에 관련된 DNA 서열의 덩어리
	- 프로모터
	- 작동자: 조절 단백질이 달라붙는 DNA 서열
		- 조절 단백질은 해당 오페론과는 별개의 유전자에 의해 암호화됨
		- 억제자: 조절 단백질 중의 하나로 활성화되면 몇몇 오페론의 유전자들이 발현되는 것을 막음
		- 활성자: 조절 단백질 중의 하나로 활성화되면 몇몇 오페론의 유전자들이 발현되도록 함
	- lac 오페론
		- 젖당 대사에 관련된 세 유전자는 lacZ, lacY, lacA 순으로 서로 인접하여 배열되어 있음
		- 작동자는 프로모터와 lacZ 유전자 사이에 존재함
		- lac 억제자는 lac 오페론과 분리된 조절 유전자인 lacI에 의해 암호화됨
			- 배지에 젖당이 없을 때 lac 억제자가 작동자에 결합
			- RNA 중합효소가 프로모터에 결합하는 것을 방해함
		- 젖당이 세포 내로 들어감
		  -> 젖당 일부가 알로락토스로 전환
		  -> lac 억제자에 결합하여 형태를 변화시킴
		  -> lac 억제자가 작동자 부위에 결합 X
		  -> RNA 중합효소가 프로모터에 결합 가능
- 원핵생물은 전사 중인 mRNA에서 번역이 시작될 수 있음

# 진핵생물에서 전사 시 조절
- 염색질 재조정
	- 염색질의 구조를 변화시켜서 전사에 관여하는 단백질들이 프로모터에 접근할 수 있게 함
	- 뉴클레오솜의 히스톤 단백질을 아세틸화시킴
		- 히스톤 단백질과 DNA 사이의 결합을 느슨하게 만듦
		- 프로모터에 RNA 중합효소가 접근하기 쉬워지도록 함
- TATA 상자: 전사 개시에 중요한 역할을 수행하는 프로모터에 포함된 DNA 서열
	- 전사인자 단백질이 TATA 상자 인식하고 달라붙음
	- 전사인자 단백질이 RNA 중합효소2를 끌어옴
- 프로모터 근접요소
	- 전사 속도를 증가시키는 조절 체계의 일부
- 증폭자
	- 멀리 떨어진 위치에서, 유전자가 가능한 범위 내 최대 속도로 전사되는지 결정하는 조절 서열
- 전사의 활성
	- 일반전사인자가 TATA 상자 위치에 붙음
	- RNA 중합효소2를 끌어옴
	- 활성자
		- 유전자들의 발현을 조절하고 전사 속도 증가시킴
		- 프로모터 근접 요소들에 붙음
		- 일반전사인자들과 직접 상호작용하여 전사 개시를 촉진함
		- 특정 종류 세포, 특정 시기에 존재하는 특정 활성자에 의해 해당 세포 내 주로 전사될 유전자를 결정함
	- 억제자
		- 활성자들과 반대되는 역할
		- 활성자들이 붙는 조절 서열(ex. 증폭자에 존재하는)에 붙어서 활성자들이 붙는 것을 막기도 함

- DNA 메틸화 반응
	- 메틸기가 효소에 의해 DNA의 사이토신 염기에 붙게 됨
		- 유전자의 전사 방해
	- 넒은 구역 또는 염색체 전체를 침묵시키기도 함
	- ex) 바소체: 염색체 상의 모든 유전자가 비활성화됨

# 진핵생물에서 전사 후 조절
- 전구체 mRNA 가공과정에서의 변이
	- 선택적 이어맞추기
		- 서로 다른 조합의 엑손과 인트론을 제거
		- 하나의 전구체 mRNA에서 여러 mRNA가 만들어짐
		- 다양한 아미노산 서열을 가진 단백질들로 번역됨
- mRNA 분해 속도의 변이
	- 프로락틴
		- 카세인 mRNA의 반감기를 늘림
			- 카세인: 쥐의 유선에서 합성되는 젖에 들어 있는 단백질
		- 젖이 만들어지도록 유선을 자극함
- miRNA에 의한 유전자 발현의 조절
	- 단백질을 암호화하지 않는 유전자에서 전사됨
	- miRNA의 전구체 RNA가 만들어짐
	- 효소의 작용을 거쳐 miRNA가 됨
	- 단백질 복합체에 붙어 있는 채로 상보적인 서열을 가진 mRNA에 붙음
	- RNAi (RNA 간섭)
		- 복합체에 존재하는 단백질들이 miRNA가 붙은 mRNA를 자름
		  -> miRNA, mRNA가 이중가닥을 이루어 리보솜에 의해 번역되지 못함
		- siRNA (작은 간섭 RNA)도 RNAi를 일으킴
			- 핵에 존재하는 유전자들로부터 만들어지지 않음
