# ⭐ **OpenAI API Key 발급 방법

## ✔ 1단계 — OpenAI 사이트 접속

여기로 들어가:

👉 [https://platform.openai.com/](https://platform.openai.com/)

로그인이 안 되어 있다면 로그인 또는 회원가입.

---

## ✔ 2단계 — “API Keys” 메뉴로 이동

로그인 후 좌측 메뉴에서:

```
Developer → API keys
```

또는 바로:

👉 [https://platform.openai.com/settings/organization/api-keys](https://platform.openai.com/settings/organization/api-keys)

이 페이지에서 API 키를 만들 수 있어.

---

## ✔ 3단계 — “Create new secret key” 버튼 클릭

버튼 모양은 이렇게 생김:

```
[ Create new secret key ]
```

누르면 팝업이 뜨고 새로운 키가 생성된다.

---

## ✔ 4단계 — 생성된 키 복사

키 형태는 이렇게 생김:

```
sk-xxxxxx...xxxx
```

⚠ **중요:**  
이 키는 단 한 번만 보여준다.  
절대 다시 확인할 수 없으니 바로 복사해서 `.env`에 넣어야 한다.

---

# ⭐ 비용은?

### ✔ OpenAI는 **완전 무료는 아님**

하지만  
처음 가입하면 **무료 크레딧(보통 5달러~ 무료 trial)**이 제공돼서 체험은 가능해.

그 뒤부터는 사용량 만큼 과금된다:

- GPT-4o: 저렴함
    
- GPT-4o-mini: 매우 저렴함 (RAG 강의 프로젝트에 충분)
    
- OpenAI embedding도 매우 싸서 부담 없음
    

---

# ⭐ 키를 넣어야 하는 위치 (.env)

키를 복사했다면 `.env` 파일에 이렇게 넣어:

📁 `.env`

```
OPENAI_API_KEY=sk-xxxxxxxxxxxxxxxxxxx
```

그 다음 Docker를 재시작:

```bash
docker compose down
docker compose up --build -d
```

---
