## 임베딩

- 자연어, 이미지, 사용자 행동 같은 비정형 데이터를 고차원의 숫자 벡터로 변환한 것
- 유사도 비교나 분류를 가능하게 함
- 임베딩 벡터 공간에서 "강아지"와 "개"는 가깝게 위치
	- 어떻게?
- 활용 사례:
	- 검색: 질의와 문서 간 의미적 유사도 비교
	- 추천 시스템: 사용자 행동 패턴을 벡터화
	- NLP 모델 입력값: BERT, Word2Vec, KoBERT 등

## SageMaker Processing

- 개념: AWS SageMaker에서 제공하는 데이터 전처리, 후처리, 평가용 관리형 서비스
- 특징:
	- 학습 전 단계에서 데이터 정제, 특징 추출
	- 학습 후 단계에서 모델 평가, 결과 분석
	- 인프라 세팅 없이 컨테이너 기반 실행 환경 제공
- 장점:
	- EC2 인스턴스 관리 불필요
	- 분산 처리 지원
	- SageMaker 파이프라인과 쉽게 연동됨
- 예시:
	- CSV -> Parquet 변환
	- 이상치 제거, 정규화, 표준화
	- 학습된 모델 성능 리포트 생성

## OpenSearch

- Elasticsearch에서 파생된 오픈소스 검색 및 분석 엔진
- AWS에서 관리형 서비스로도 제공
- Full-text search: 키워드 기반 검색 (ex. 뉴스 기사 검색)
- 벡터 검색: 임베딩을 기반으로 의미적 유사도 검색 (ex. "강아지 사진"과 비슷한 이미지 검색)
- 실시간 로그 분석: Kibana(OpenSearch Dashboards)로 시각화

## Amazon SageMaker AI

- ML 모델을 쉽게 학습, 구축할 수 있는 AWS의 완전관리형 서비스
- ML 파이프라인의 다양한 단계를 관리하는 여러 도구와 서비스 제공
	- SageMaker Studio
		- 전체적인 개발 환경(IDE)의 대시보드
		- ML 모델을 구축, 학습, 튜닝, 배포하는 데 필요한 모든 기능을 제공
	- SageMaker Notebooks
		- Jupyter 노트북을 기반으로 한 인터페이스
		- 데이터를 분석하고 모델을 훈련하는 데 사용
		- SageMaker Studio Notebooks: SageMaker Studio 내부에서 실행
		- SageMaker Notebook Instances: 독립적인 Jupyter Notebook 인스턴스
	- SageMaker Experiments
		- 하이퍼파라미터, 데이터셋, 코드 버전을 비교하고 분석하는 서비스
		- SageMaker Studio 내에서 설정
	- SageMaker Training
		- ML 모델을 훈련시키는 서비스
		- 훈련 데이터를 준비하고, 모델을 훈련시키며, 최적의 모델을 찾는 데 필요한 다양한 도구와 리소스 제공
	- ...
