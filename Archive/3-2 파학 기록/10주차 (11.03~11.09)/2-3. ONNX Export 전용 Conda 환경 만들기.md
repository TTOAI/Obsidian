
- 가상 환경 생성
```
conda create -n onnx-export python=3.10
conda activate onnx-export
```

- requirements.txt
```
torch==2.3.0
torchvision==0.18.0
transformers==4.38.0
onnx==1.15.0
boto3
kobert @ git+https://github.com/SKTBrain/KoBERT.git@fcd729f2f4b37858f333597c0782388ada51eb5f
```

- 라이브러리 설치 및 실행
```
pip install -r requirements-onnx-export.txt 

python export_imfact_models_to_onnx.py

python upload_onnx_to_s3.py
```

- (참고) np.bool 에러
	- numpy 1.20부터 `np.bool` 제거됨
	- MXNet 1.7 코드는 `np.bool`을 사용하고 있음
	- 그래서 MXNet import 시 바로 죽음
```
AttributeError: module 'numpy' has no attribute 'bool'
```
