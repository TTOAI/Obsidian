- 돼지고기 삼겹살, 목살 요리

- 닭가슴살, 닭안심 구이
	- 고기 + 후추 + 소금 + 밀가루 + 레몬즙
	- 구우면서 레몬즙 추가

- 남궁민 벌크업 식단
	- 고기 굽기
	- 밥 + 스크램블된 계란 넣고 계란 익히기
	- 케찹 뿌리기

- 스크램블 에그
	- 계란 3개 + 우유(생크림) 1/2컵 + 소금, 설탕 적당히, 버터 2cm 정도 두께 녹이고 뒤집개로 바닥 긁으면서 익히기

- 오트밀 요리
	- 닭죽 느낌: 오트밀 + 고기 + 소금 + 물, 끓이면서 물 추가, 참기름 + 후추 첨가
	- 토마토 리조또 느낌: 양파 + 다진마늘 볶기,  토마토소스 + 우유 넣고 볶기, 오트밀 넣고 볶기, ==치즈== 추가하고 볶기
	- 김치참치죽: ==김치== 세 스푼 자르기 + 참치 한 캔 + 오트밀 + ==고춧가루== + 물 한 컵  + 간장 한 스푼 + 다진마늘 + 후추 + 계란 넣고 전자레인지, 참기름

- 연어회
	- 소금에 절여서 실온 1시간(이슬 맺힐 정도까지)
	- 흐르는 물에 담가서 씻기
	- 식초:물 1:2에 담가서 30분
	- 양파 물에 담가서 매운 맛 빼기
