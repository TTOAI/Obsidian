- 프로젝트별 라이브러리 설정의 독립성 유지하기 위해 가상환경을 설정함
- 가상환경 설정에는 venv를 사용함

- venv 설정 방법
```bash
python -m venv ./.venv
source ./.venv/Scripts/activate
```

---

- 현재 환경에 설치된 모든 패키지 목록과 버전 정보를 파일로 저장
```bash
pip freeze > requirements.txt
```

- 다른 환경에서 재현
```bash
pip install -r requirements.txt
```

### WSL Ubuntu
```
# 가상환경 생성
python3 -m venv .venv

# 가상환경 활성화
source .venv/bin/activate

# 패키지 설치 (이제 이 안에만 설치됨)
pip install torch transformers

# 비활성화
deactivate
```
