
### environment.yml

```yml
name: imfact_infer_env
channels:
  - conda-forge
  - defaults
dependencies:
  - python=3.10
  - boto3=1.37.38
  - pip:
      - numpy==1.23.5
      - kobert @ git+https://github.com/SKTBrain/KoBERT.git@fcd729f2f4b37858f333597c0782388ada51eb5f
      - kobert-tokenizer @ git+https://github.com/SKTBrain/KoBERT.git@fcd729f2f4b37858f333597c0782388ada51eb5f#subdirectory=kobert_hf
```

- (참고) Conda 환경 재현
```shell
conda env create -f environment.yml
```

### Dockerfile

```dockerfile
FROM python:3.10-slim
  
WORKDIR /app
  
COPY . .
  
RUN pip install --no-cache-dir -r requirements.txt
  
CMD ["python", "imfact_batch_infer.py"]
```
