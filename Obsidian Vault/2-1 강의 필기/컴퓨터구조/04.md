
- 요점 정리
	- machine code = 32-bit instruction words
	- MIPS Instruction Formats에서 funct는 16진수임에 주의
		- ex) 0x22 = 10 0010
	- R-format
		- rs: 1st source register, rt: 2nd source register, rd: destination register
	- I-format
		- immediate arithmetic instructions
			- rs: source register, rt: destination register, constant: signed integer
		- load/store instructions
			- rs: source register containing the base address
			- rt: destination/source register
