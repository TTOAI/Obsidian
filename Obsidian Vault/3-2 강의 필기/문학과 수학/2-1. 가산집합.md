- 윤동주의 '별 헤는 밤'
	- '하늘과 바람과 별과 시'에 수록된 시

- 진짜 별을 헤아릴 수 있을까?
- '헤아린다'는 말의 수학적 의미

- 대응
	- 두 집합 또는 수학적 대상에 어느 두 개가 서로 짝이 되어 상대하는 경우
	- 일대일 대응: 집합 X의 원소와 집합 Y의 원소가 중복 없이 정확히 하나씩 대응

- 헤아린다
	- 헤아리고 싶은 대상의 집합과 자연수 집합의 일부가 서로 일대일 대응

- 어떤 집합이 자연수의 부분집합과 일대일 대응을 할 수 있음
	- -> 헤아릴 수 있는 집합 = 가산집합

- 별은 과연 자연수의 부분집합과 대응을 시킬 수 있을까?
- 별은 무한히 많을 수 있음

- 무한대
	- 수가 아니라 상태, 개념적 대상
	- 어떤 수를 생각해도 그 수보다 더 큰 상태

- 힐베르트의 호텔
	- 무한대의 독특한 성질 설명
	- '무한 호텔 역설'이라고 함
		- 무한대가 특정 수가 아니라 개념이기 때문에 가능
	- 무한히 많은 방이 있는 호텔에 무한히 많은 손님을 태운 버스가 무한히 많이 옴
	- m번째 버스에 탄 손님들 중에서 n번째에 있는 손님 = (m, n)
	- 손님들이 힐베르트 호텔에 묵는다 -> 차례대로 자연수에 대응할 수 있는가
	- 자연수 좌표를 대각선으로 차례대로 읽어나가면 자연수에 일대일 대응시킬 수 있음

- 정수:
	- 음의 정수, 0, 양의 정수(자연수)로 이루어짐

- 유리수:
	- 분자와 분모가 모두 정수인 분수꼴
	- 분모는 0이 될 수 없음
	- 정수의 일반적인 개념

- 무한히 많은 수의 손님
	- 전부 음수로 간주 -> 정수와 자연수 대응
- 무한대의 손님을 태운 무한대의 버스
	- 유리수를 자연수에 대응

- 헤아릴 수 없는 집합 -> 자연수와 대응이 불가능한 집합
