# 서브쿼리
- GROUP BY절 제외 사용 가능

- 위치에 따라
	- 스칼라 서브쿼리
		- SELECT절
		- 서브쿼리 결과 -> 컬럼
	- 인라인뷰
		- FROM절
		- 서브쿼리 결과 -> 테이블
	- WHERE절 서브쿼리
		- 가장 일반적인 서브쿼리
		- 서브쿼리 결과 -> 상수항

- WHERE절 서브쿼리 종류 (리턴 데이터 형태)
	- 단일행 서브쿼리
		- 1개의 행 리턴
	- 다중행 서브쿼리
		- 여러 행 리턴
		- 서브쿼리 결과 하나로 요약
			- ex. MIN(), MAX()
		- 다중행 서브쿼리 연산자 사용
			- ex. ANY(), ALL()
	- 다중컬럼 서브쿼리
		- 메인 쿼리와의 비교 컬럼이 2개 이상
	- 상호연관 서브쿼리
		- 메인쿼리와 서브쿼리의 비교를 수행
		- WHERE절의 조건으로 메인쿼리와 서브쿼리에서 비교할 컬럼을 특정
			- ex. E1.DEPTNO = E2.DEPTNO

- 인라인뷰
	- 쿼리 안의 뷰 형태로 테이블처럼 조회할 데이터를 정의하기 위해 사용
	- 서브쿼리 결과를 메인 쿼리의 어느 절에서도 사용할 수 있음
	- 인라인뷰의 결과와 메인쿼리의 테이블을 조인할 목적으로 주로 사용
	- 모든 연산자 사용 가능

- 서브 쿼리 주의 사항
	- 서브 쿼리절에 ORDER BY절 사용 불가

# 집합 연산자
- 합집합
	- UNION
		- 중복된 데이터는 한 번만 출력
		- 내부적으로 정렬 수행
		- 중복 데이터가 없을 경우 UNION ALL 사용
	- UNION ALL
		- 중복된 데이터도 전체 출력
- 교집합
	- INTERSECT
- 차집합
	- MINUS
- 그룹함수
	- 한 컬럼만 전달
	- NULL 무시하고 연산
