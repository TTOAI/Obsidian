# 3. 백엔드 설계
- RESTful API 설계
- 각 기능에 필요한 엔드포인트 정의
- 데이터 처리 로직을 spring으로 구현
- **Controller** 작성: 각 엔드포인트를 정의하여 요청을 처리하는 곳
- **Service** 레이어 구현: 비즈니스 로직을 처리하는 로직을 작성
- **Repository** 작성: 데이터베이스와 상호작용을 위한 코드 작성

# 4. 프론트엔드 설계
- UI/UX 디자인 구체화
- 컴포넌트 구조 설계
- API 통신이 필요한 부분을 백엔드 담당자와 협의
	- 페이지 레이아웃 및 데이터 바인딩 방식을 결정
- 각 페이지에 필요한 상태 관리와 데이터 흐름을 정의
- React Router 등을 사용하여 페이지 이동을 관리

# 5. 깃과 깃허브를 이용한 협업 설정
