# 정보의 흐름과 뉴런
### 신경계에서의 뉴런과 그들의 구성
- 뉴런에서 전기신호 이동 방향
	- 수상돌기 -> 세포체(핵) -> 축삭 -> 축삭말단
- 신경아교세포
	- 뉴런에 영양소를 제공하고 뉴런을 지지하는 비신경성 세포
	- 성상세포
		- 별모양
		- 뉴런을 물리적으로 지지
		- 그들을 둘러싸고 있는 세포간질액의 이온 농도를 유지하도록 도움
	- 희소돌기아교세포
		- 중추신경계에 존재
		- 미엘린 수초 형성
		- 한 개의 세포가 여러 축삭을 감쌀 수 있음
	- 슈반세포
		- 말초신경계에 존재
		- 젤리 롤 모양 미엘린 수초 형성
		  -> 높은 지질함량
		  -> 전기적 절연체
		- 랑비에 결절
			- 슈반세포 사이의 틈
- 시냅스
	- 축삭말단 (시냅스전세포)
	  -> 신경전달물질
	  -> 수상돌기 (시냅스후세포)
	- 전기시냅스
		- 시냅스전/후세포의 세포막이 직접 접촉
	- 화학시냅스
		- 시냅스전/후세포의 원형질막이 시냅스틈에 의해 분리되어 있음

### 뉴런에 의한 신호 전도
- 세포 안팎의 Na+, K+ 불균등 분포
  -> 세포 외부는 양전하, 내부는 음전하로 분리됨
  -> 막전위를 가짐
- Na+/K+ 능동수송펌프로 에너지를 사용하여 Na+ 밖으로, K+ 안으로 이동
- 휴지전위: 자극받지 않은 뉴런의 변하지 않은 막전위 (약 -70mV)
- 활동전위
	- 전기충격
	  -> 탈분극: 양전하가 뉴런 안으로 들어옴
	  -> 역치전위 도달 전: 탈분극이 서서히 진행됨
	  -> 역치 도달 후: 활동전위 발생, 막전위가 급격히 상승
	  -> 약 +30mV 도달 후 약 -80mV 까지 떨어짐 (과분극)
- 모든 자극들은 뉴런의 탈분극을 일으킴
- 역치 이상의 자극만 활동전위를 일으킴
  -> 실무율의 원리
- 한 번 시작된 막전위의 변화는 자극의 세기와는 관계없이 일어남
- 전압개폐성 이온통로: 막전화의 변화에 의해 열고 닫히는 세포막에 박혀 있는 단백질
- 국부마취제: Na+통로에 결합하여  이온 이동 능력을 방해함
- 다발성 경화증: 미엘린이 점점 사라짐

### 화학시냅스에서의 신호전달
- 리간드개폐성 이온통로: 시냅스후세포막에 있고 신경전달물질에 의해 활성화됨
- 신경전달물질
	- 아세틸콜린: 높은 수준의 뇌기능 수행하는 뉴런들 사이에서 작용
