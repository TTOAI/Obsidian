
- 생물계
	- 생물권: 모든 생태계
	- 생태계: 공유하는 물리적 환경, 상호작용
	- 군집: 같은 지역 점유, 모든 종
	- 개체군: 같은 지역 점유, 같은 종
	- 다세포 생물: 의존하는 세포들
	- 세포: 독립적, 가장 작은 단위

- DNA에서 RNA로 복제
- RNA는 단백질 분자의 제조 명령

- 이온결합은 다방향성, 전기인력에 의해, 다양한 세기
- 공유결합은 전자에 의해

- 극성 분자는 끼리끼리, 비극성 분자를 몰아냄
- 극성 분자는 친수성
- 비극성 분자는 소수성

- 수소 결합은 양전하, 음전하를 띠는 원자 사이의 약한 전기적 결합
- 수소 결합은 수소와 산소, 질소, 황 사이에서 생김
- 수소 결합은 물 분자 간에 쉽게 형성 (수소 원자 + 산소 원자)
- 단백질 분자를 나선 모양으로 안정화시킴
- 수소 결합은 격자 구조
	- 얼음의 낮은 밀도로 인해 물이 위부터 얼음
	- 세포막의 인지질 이중막 구조
		- 물과의 접촉을 최소화하려고 소수성 꼬리끼리 뭉침
		- 외부의 물로부터 보존
	-  높은 비열, 높은 기화열(체온 냉각), 응집력
- 물은 극성 물질을 녹이는 용매 역할

- 유기분자: 탄소
- 무기분자: 이산화탄소 + 탄소 X

- 탄수화물: 탄수산 1:2:1
- 단당류: 알파 포도당(녹말), 베타 포도당(섬유소)
- 이당류: 엿당(포포), 설탕(포과), 젖당(포갈)
- 다당류: 녹말, 글리코젠, 섬유소

- 지질: 극성 X, 탄화수소 분자
	- 중성지질, 인지질, 스테로이드
- 중성지질
	- 트라이글리세라이드: 글리세롤 + 3지방산
		- 비극성
		- 탄수화물 2배 에너지
	- 지방산
		- 포화: 단일결합만 -> 지방(상온 고체), 직선 형태
		- 불포화: 이중결합 존재 -> 기름(상온 액체), 구부러진 형태
- 인지질
	- 생체막의 기본 구조
	- 극성 단위 + 인산기(극성) + 글리세롤 + 2지방산(비극성)
	- 극성 환경에서 극성 말단(인산기)만이 물에 노출됨
	- 비극성 말단(지방산)은 모이면서 물을 몰아냄
- 스테로이드
	- 4개의 탄소고리
	- 소수성

- 단백질
	- 20종류의 아미노산, 펩타이드 결합 -> 폴리펩타이드
	- 측기(R)의 형태, 아미노산 숫자와 형태에 따라 다양함
	- 1차 구조: 아미노산의 서열, 단백질의 형태와 기능 결정
	- 2차 구조: 아미노산 사슬의 꼬임과 회전
	- 3차 구조: 삼차원적 모양, 입체 구조, 형태
	- 4차 구조: 폴리펩타이드 사슬들의 배열

- 뉴클레오타이드
	- 질소함유 염기
		- 피리미딘(고리 1) -> 유라실, 타이민, 사이토신
		- 퓨린(고리 2) -> 아데닌, 구아닌
	- 5탄당
		- 디옥시리보스, 리보스
		- 2' 탄소에 붙는 화학기
			- H -> 디옥시리보스 (DNA 형성)
			- OH -> 리보스 (RNA 형성)
	- 인산기

- 핵산
	- 뉴클레오타이드가 반복적으로 결합
		- 탄소 원자 사이에 인산기
	- DNA: 디옥시리보스 + A, T, G, C
		- 2중 나선
		- A + T: 수소 결합 2개
		- G + C: 수소 결합 3개
	- RNA: 리보스 + A, U, G, C

- 원핵생물: 진정세균 영역, 고세균 영역
	- 핵양체: 세포질로부터 분리시키는 막 없음
	- 세포벽이 끈적한 협막으로 덮혀 있음
	- 리보솜: 아미노산 조립해서 단백질 만드는 소기관

- 진핵생물: 진핵생물 영역 (원생생물계, 균계, 동물계, 식물계)
	- 핵: 세포질로부터 막으로 분리
		- 핵막: 이중막
		- 핵공: RNA, 단백질 통로
		- 염색질: DNA 분자 + 단백질 집합체
		- 염색체: 단백질과 결합된 하나의 DNA 분자
		- 인: rRNA 분자들 + 단백질 -> 리보솜 단위체 형성

- 내막계: 세포 내 막으로 된 주머니의 집합체

- 소포체
	- 조면
		- 리보솜이 박혀 있음
			- mRNA 정보 이용, 단백질 합성
		- 소포체 내강에서 단백질이 최종 형태로
		- 낭을 통해 골지복합체로
	- 활면
		- 지질(원형질막 구성성분) 합성
		- 독성물질 순화 효소 가짐
- 골지복합체
	- 단백질에 화학적 변형
	- 단백질 이동 조절
	- 세포에서 분비되는 단백질, 원형질막에 박히는 단백질 등을 분리
- 리소좀
	- 가수분해효소 가짐
	- 자가소화작용
	- 식균작용
- 미토콘드리아
	- 세포호흡: 크리스테와 기질에서 일어남
	- 내막의 굴곡: 크리스테
	- 기질: 안쪽 부분
- 세포골격
	- 미세소관: 튜불린, 세포소기관 고정과 이동
	- 중간섬유: 중간 정도의 굵기
	- 미세섬유: 액틴, 세포질 운동, 아메바 운동, 근육세포의 수축
	- 세포분열할 때: 염색체 이동 - 미세소관, 세포질 분리 - 미세섬유

- 엽록체
	- 외막, 내막: 스트로마를 감쌈
	- 틸라코이드: 납작한 주머니 모양, 엽록소 분포
	- 그라나: 틸라코이드가 쌓임
	- 엽록소: 빛 흡수
- 중심 액포
	- 구조적 지지, 세포 성장에 다양한 역할, 노폐물 포함 물질 보관, 저장
- 세포벽
	- 섬유소(셀룰로오스)로 구성
	- 식물세포 지지, 보호

- 세포막
	- 지방(인지질, 스테롤)과 단백질로 구성
	- 인지질
		- 극성 인산기: 표면, 비극성 지방산: 세포막 안쪽
	- 특성화된 기능을 가지는 단백질 그룹들
	- 당지질
	- 당단백질
	- 막 지질, 막 단백질: 세포막 표면에 극성을 띠는 탄수화물 그룹과 연결
	- 유동(인지질 2중층) 모자이크(단백질) 모형
	- 소수성(비극성) 분자는 자유롭게 통과 가능
	- 친수성, 전하를 띤 원자나 분자: 선택적 투과성
		- 수동 수송: 에너지 소비 X
			- 단순확산(소수의 물질)
				- 농도기울기에 따라
				- 지질 부분 통과
				- 비극성 무기질 가스 (산소, 질소, 이산화탄소)
				- 비극성 유기물질 (스테로이드 호르몬)
			- 촉진확산
				- 농도기울기에 따라
				- 수송 단백질의 도움
					- 채널 단백질, 운반체 단백질
		- 삼투: 수동적 물 수송
		- 능동 수송
			- 농도기울기에 반하여 물질이 통과
			- 1차 능동수송
				- 물질 운반 단백질이 ATP 분해, 에너지 제공
			- 2차 능동수송
				- 1차에 의한 이온의 농도기울기를 에너지원으로 사용
