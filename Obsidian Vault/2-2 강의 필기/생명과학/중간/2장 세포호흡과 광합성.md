
(전체적인 원리 - 전자의 흐름 순서 위주)

- 운동에너지 (전기에너지, 복사에너지, 열에너지, 소리와 움직임의 에너지)
- 위치에너지 (화학에너지, 핵에너지, 중력에너지, 저장된 역학에너지)

- 이화작용
	- 당분자 분해 (위치에너지 -> 운동에너지)
	- 자유에너지 △G 음수

- 동화작용
	- 에너지를 이용해 기질을 복합체 분자로 조립
	- 자유에너지 △G 양수

- ATP 사용: 흡열반응 <-> 발열반응
	- ATP: 질소염기 + 5탄당 리보스 + 3인산기
	- ATP = ADP + 무기인산(P)
- 주요 생물학적 반응: ATP 형성 <-> 가수분해
- ATP 가수분해를 통해 구동: 동화반응, 단백질 활성, 용질 수송

# 효소
- 활성화 에너지 감소, 반응 촉진
- 자발적 반응도 활성화 에너지 필요
- 효소 특이성
- 기질
- 활성부위
- 보조인자
	- 무기물 보조인자
	- 유기물 보조인자 (조효소)
- 좁은 범위의 온도, pH에서 최고 활성도

- 산화 환원 반응
	- 산화: 전자를 잃음
	- 환원: 전자를 얻음
	- 반응물 산화 -> 잠재에너지 떨어짐 -> 화학에너지 방출

# 세포호흡
- 유기분자가 산화하는 반응
- 반응물로 포도당 사용
- ADP + P(인산기) -> ATP를 만드는 반응 (인산화)
	- 산화를 통해 방출된 에너지로 ATP 합성 -> 산화적 인산화

- 1단계: 해당과정
	- 세포액에서 일어남
	- 에너지 요구 단계 + 에너지 방출 단계
	- ![[Pasted image 20241015135820.png]]
	- 기질수준 인산화: 기질의 인산기 + ADP -> ATP
	- 피루브산이 생성될 때 2ATP 생성
- 2단계: 피루브산 산화, 시트르산 회로
	- 미토콘드리아의 기질에서 일어남
	- 피루브산 산화
		- ![[Pasted image 20241015135828.png]]
	- 시트르산 회로
		- ![[Pasted image 20241015135835.png]]
	- 탄수화물, 지질, 단백질은 산화과정의 전자공급원으로 작용
- 3단계: 전자전달계
	- NAD+, FAD 등의 운반체에 의해 운반되는 고에너지 전자
		  -> 전자전달계로 들어가서 많은 양의 ATP를 만드는 데 사용됨
	- 미토콘드리아 내막에 전자전달계와 ATP 합성효소가 위치함
	- 전자전달계
		1. 운반체가 고에너지 전자를 산소로 전달
		2. 운반체를 통한 전자의 흐름이 자유에너지 방출
		3. 미토콘드리아 내막에 위치한 전자운반체, 유비퀴논, 복합체
		  -> 복합체 사이에서 전자운반체와 유비퀴논이 전자 전달
		  -> 전자 흐름에 의한 에너지를 사용하여 H+ 능동 수송
		  -> H+가 기질에서 막간구획으로 펌프질됨
		4. 막을 경계로 한 양성자와 전압의 기울기
		  -> 양성자구동력(저장에너지) 형성
		  -> ATP 합성을 위한 에너지 제공
		  -> H+가 ATP 합성효소를 통과하며 ATP 합성이 일어남

- 포도당 -> 이산화탄소 + 물 로 산화되며 총 32개의 ATP 분자를 만듦
- 세포호흡은 포도당의 화학에너지 30% 이상을 ATP에 보존
	- 나머지 열로 방출 -> 체온 유지

# 발효
- 산소가 부족할 때: NADH에 의해 운반되는 전자가 발효과정에 이용
	- 전자 -> 전자전달계 (X)
	- 전자 -> 유기물의 전자수용체로 전달
		- NADH -> NAD+로 산화
		  -> 해당과정 계속 가능
- 젖산 발효
	- 동물 근육세포의 세포액에서 일어남
	- 피루브산 -> 젖산으로 환원
- 알코올 발효
	- 미생물에서 일어남 (단세포 균류 - 효모)
	- 피루브산 -> 에틸 알코올 + 이산화탄소로 환원


- 독립영양생물: 무기질 원료 -> 유기분자를 생성하는 생물
	- 광독립영양생물: 에너지원으로 빛 사용
- 종속영양생물: 유기분자를 필요로 하는 소비자와 분해자

# 광합성
- 1단계: 광의존반응
	- 물 + 빛에너지 -> 화학에너지 (ATP, NADPH 형태) + 산소
	- ATP: 식물 주요 에너지원
	- NADPH: 고에너지 전자 운반
- 2단계 광독립반응 (= 캘빈회로)
	- CO2 고정
		- CO2 (무기질) + 전자 + 수소이온 -> 탄수화물 (유기 형태) + 산소
	- 식물, 조류, 남세균: 물이 전자와 수소이온의 공급원 역할

- 진핵 생물의 광합성은 엽록체에서 일어남
- 엽록체
	- 3개의 막
		- 외막
		- 막간구획
		- 내막
		- 스트로마(유동액)
		- 틸라코이드막

- 광의존반응
	- 광의존반응 분자(색소, 전자운반전달체, ATP 생성효소): 틸라코이드막, 스트로마 라멜라에 분포
	1. 빛 흡수 과정
		- 엽록체의 틸라코이드막: 엽록소, 카로티노이드 색소 분자 존재
		- 광계1, 2: 빛 흡수 색소 + 단백질 + 다른 분자
			- 안테나 복합체
				- 엽록소 색소 + 카로티노이드 분자
			- 반응중심
				- 단백질과 결합된 특별한 엽록소 a 한 쌍
		- 빛에너지가 안테나 복합체에 의해 흡수
		- 유도공명 에너지 전달
			- 들뜬 전자의 에너지 -> 인접한 분자로 이동, 반응중심 엽록소 도달
		- 반응중심에서 에너지는 들뜬 전자의 형태로 포획
		  -> 1차 수용체 분자로 이동 (전자전달계)
	
	2. NADPH와 ATP 생성
		- 물이 분해되면서 얻어진 전자 -> 광계 2로 들어감
		- 광계 2는 빛을 흡수하여 전자를 들뜨게 함
		- 빛에너지에 의해 들뜬 전자
		  -> 전자운반체로 구성된 전자전달계를 통과하면서 전달됨
		
		- 물로부터 온 전자
		  -> 광계2: 빛에너지에 의해 들뜸
		  -> 전자전달계: 에너지 준위가 떨어지면서 흐름
		  -> 자유에너지 방출
		  -> 양성자(H+)가 틸라코이드 내강으로 펌핑
		  -> 막을 경계로 H+ 농도기울기를 형성
		  -> H+ 농도기울기는 ATP 합성의 에너지원
		  -> 광계1로 이동: 빛에너지에 의해 들뜸
		  -> 전자전달계: 마지막 전자수용체인 NADP+까지 전달
		  -> NADP+ 환원효소: NADP+를 NADPH로 환원시키고 양성자 방출
		  -> 막을 경계로 만들어진 양성자, 전하 기울기
		  -> 양성자 구동력(저장에너지) 형성
		  -> ATP 생성효소가 ATP 합성하도록 에너지 제공
	
	- 광인산화: 빛에너지를 받은 전자 흐름과 짝을 이룬 ATP 합성
		- 산화적 인산화와 비슷

- NADPH: 모든 진핵세포에서 동일한 주요 역할
	- 환원이 필요한 합성반응에 고에너지의 전자를 전달
	  -> CO2 고정 과정: 광독립반응에서 일어남

- 광독립반응
	- 광독립반응: 스트로마
	- NADPH: 이산화탄소를 탄수화물, 다른 유기분자로 고정하는 환원력 제공
	- 필요한 추가의 에너지: 광의존반응에서 생성된 ATP
	
	- 캘빈회로는 3단계로 구성
	- 1단계: 탄소고정 단계
		- 이산화탄소 + 5탄당 리불로스(RuBP) 1,5-2인산 -> 3탄소 분자 3인산글리세르산(3GPA) 2개
			- RuBP 카복실화효소/산소화효소에 의해 촉매
	- 2단계: 환원반응
		- NADPH + ATP + 3PGA -> G3P
		- 캘빈회로가 몇 회 순환된 후 포도당 생성
	- 3단계: 재생
		- G3P + ATP -> RuBP
		- RuBP 재생하기 위해 ATP의 인산기 사용

- 광호흡
	- 이산화탄소 대신 산소가 루비스코의 활성부위에 결합
	  -> 루비스코: 카복실화효소 대신 산소화효소로 작용
	- 산소화효소 반응: O2 흡수, CO2 방출 -> 광호흡
	- 독성인 글리콜산 생산 -> CO2 방출 경로들에 의해 제거
	- 식물의 생장 저해 -> 예비반응 진화
		- C4회로
			- 고농도의 산소에 영향 받지 않는 (산소화효소 활성이 없는) 다른 카복실화효소에 의해 CO2를 먼저 고정
			- CO2 -> 옥살로아세트산 -> 말산
			- O2 농도가 높을 때 작동
			- O2 농도가 낮을 때 말산이 CO2를 방출
			  -> CO2는 캘빈회로로 들어감
	- 높은 온도는 광호흡의 수준을 증가시킴
		- 높은 온도 -> 광합성 속도 증가 -> 기공을 열게 함 -> 물 방출 -> 탈수 현상 -> 기공을 닫게 함 -> 산소 축적 -> 광호흡 속도 증가

# C4 식물의 광호흡 극복
- C4와 캘빈회로를 다른 장소에서 작동시켜 광호흡 극복
	- 옥수수
		- O2가 풍부한 엽육세포에서 C4 회로가 일어남
		- C4회로로 말산이 생성됨
		- O2가 덜 풍부한 관다발초세포로 확산
		- 말산이 CO2로 전환
		- 캘빈회로 일어남

- C4와 캘빈회로를 다른 시간에 작동시켜 광호흡 극복
	- CAM 식물 (예: 선인장)
	- 낮에 고온건조, 밤에 추운 지역에서 생존
	- 기공 수 적고 밤에만 열림
		- 밤에 산소 방출
		- CO2가 잎으로 들어옴
		- C4 경로에 의해 말산으로 고정, 축적, 액포에 저장
		- 태양이 떠오르고 기온이 올라감
		- 액포에서 세포질로 확산
		- CO2가 고농도로 방출
		- 캘빈회로 일어남
