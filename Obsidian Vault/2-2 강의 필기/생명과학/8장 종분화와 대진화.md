
# 종개념
- 형태학적 종개념
	- 린네의 생물분류법
		- 새로운 종을 표현하기 위해 사용
		- 눈에 보이는 해부학적 특징에 근거
	- 외형상 거의 동일한 근연종 구별 힘듦

- 생물학적 종개념
	- 종의 기능적인 성질에 중점을 둠
	- 생식력 있는 자손을 만듦 -> 같은 종
	- 개체군 유전학과 진화이론으로 종 구분
		- 같은 종의 개체군은 유전자 확산을 통해 유전물질이 섞임
		- 하나의 종 -> 하나의 거대한 유전자 풀
	- 각각의 종은 유전적으로 별개임 -> 유전정보 교환 X
	- 무성생식으로 번식하는 생명체들에게 적용 어려움

- 계통발생적 종개념
	- 관심 개체군의 진화계통수 재구성
	- 계통발생적 종
		- 계통수에서 파생된 개체군의 집단
		- 하나의 진화 역사를 공유하는 개체군

- 아종
	- 분리된 종의 개체군들은 조종 유전자나 표현형이 달라짐
	- 아종: 지리적으로 멀리 떨어진 개체군들 사이에서 쉽게 인지되는 지역적 표현형 변이
	- 종분화의 과정을 보여줌

# 생식적 격리
- 생물학적 종개념의 중심
- RIM (생식적 격리기구): 두 종의 유전자 풀이 서로 섞이지 않도록 하는 생물학적 특징
- 격리기구는 둘 이상이 동시에 작동하기도 함
- 접합 전 격리기구
	- 접합자나 수정란의 생성 이전에 작용
	- 잡종 자손을 생성되지 못하게 함
	- 생태적 격리
		- 종들이 서로 다른 서식지에 놓이게 됨
		  -> 서로 접촉할 수 없고 상호 교배도 일어나지 않음
	- 시간적 격리
		- 서로 다른 시기에 교배를 함
	- 행동적 격리
		- 많은 동물들은 교배 가능한 종을 확인하기 위해 종들 사이에 서로 다른 특수한 신호를 사용함
		- 한 종에서 사용되는 신호들은 다른 종에서는 인식되지 않음
	- 기계적 격리
		- 생식기관을 비롯한 몸의 부분에서 차이가 남
		- 많은 식물들은 특정한 꽃가루매개자만이 자신에게 접근할 수 있도록 고유의 해부학적 구조를 가짐
	- 배우자 격리
		- 한 종의 정자와 다른 종의 난자 사이의 불화합성
- 접합 후 격리기구
	- 접합자 형성 이후에 작용
	- 잡종 생존불능
		- 양쪽 부모로부터 배아 발달을 조절하는 유전자를 물려받음
		- 두 부모 종의 발달진행 과정이 서로 맞지 않아 잡종 배아는 출산예정일이 되기 전에 죽음
	- 잡종 불임
		- 부모 종의 염색체 수나 구조가 다름
		  -> 감수분열 시 적절한 쌍을 이루지 못함
		  -> 기능적인 배우자를 만들지 못함
	- 잡종 붕괴
		- 잡종 자손에서 만들어진 자손들은 높은 빈도의 염색체변이와 해로운 유전자 조합을 가지고 태어남

# 지리적 연관관계에 따른 종분화
- 이소적 종분화
	- 이소: 다른 거주장소
	- 두 집단이 지리적으로 분리되어 유전자 확산이 차단됨
	  -> 서로 다른 돌연변이, 자연선택, 유전적 부동 등을 겪음
	  -> 유전자의 차이가 축적되고 생식적으로 격리됨
- 동소적 종분화
	- 하나의 개체군 내에서 발생된 하위집단 사이에서 진화
	- ex. 사과파리
		- 숙주식물로 산사나무만 존재했음
		- 새로운 숙주종의 애벌레가 사과를 먹고 있는 것을 발견
		- 각각의 숙주종의 개체가 같은 숙주종의 구성원과 더 많이 교배
		- 숙주종 사이의 생식적 격리 발달
	- ex. 배수성과 같은 유전 현상
		- 배수성 개체는 부모 종의 개체와 교배하지 못함
		- 종분화 진행
		- 중요한 농작물, 현화식물 종의 거의 반은 배수체임

# 고생물학과 대진화
- 화석: 생물의 잔존체가 누적되는 퇴적층에 묻힐 때 지층 내에서 형성
	- ex. 규화목: 목재를 구성하는 분자가 무기질 분자로 치환되어 형성
	- ex. 원생대 무척추동물: 부드러운 조직이 미세한 퇴적층에 흔적으로 보존
	- ex. 호박 속 파리와 맵시벌: 산소의 결핍이 부패를 방지하여 보존
	- ex. 메머드 화석: 빙하 얼음 속에 보존

- 방사성 연대측정법
	- 암석의 절대적인 연대를 추정하기 위한 기술
	- 어버이 동위원소와 붕괴된 산물의 상대적인 양 측정
	  -> 동위원소의 반감기 이용해서 암석의 나이 추정
	- ex. 탄소화합물(뼈 또는 목재)로 이루어진 화석
	  -> 14C가 14N으로 붕괴되는 양 측정
	- ex. 생명체의 화석
	  -> 살아 있는 한 일정하게 유지되는 14C가 죽은 후 일정 속도로 붕괴
	  -> 14C/12C의 비를 결정하여 화석의 연대 결정

- 판구조론
	- 지질학적 과정이 지구의 물리적 환경을 변화시킴
	- 대륙이동: 맨틀 대류가 대륙을 포함하는 판을 이동
		- 지구기후의 전반적인 변화를 유도함
	- 화산폭발, 운석충돌 등의 사건들
		- 지구의 대기권과 기후를 급진적으로 변화

- 수렴진화
	- 연관이 먼 종들이 비슷한 환경에서 유사하게 보이도록 적응 진화

- 진화 계열의 재해석 - 말의 진화적인 경로
	- 마시
		- 말의 진화를 체형이 커지고, 어금니의 크기가 증가하는 등의 방향으로 후손들을 특징짓는 선형적 변화로 기술
	- 최근의 연구
		- 말 과는 다양한 형태를 갖는 수많은 진화분지를 포함함
		- 말이 직선적인 단계를 거쳐 점진적으로 변화하지는 않음

- 점진주의 이론 vs 단속평형 이론
	- 형태학적 변화의 속도 또는 시간을 기재하는 두 가지 대안적 가정
- 점진주의 이론
	- 큰변화는 시간에 따라 천천히, 연속적인 작은 변화가 축적된 결과
	- 이론에 따라 일련의 점진적 화석 발견이 예견됨
	  -> 하지만 거의 발견할 수 없음
- 단속평형 이론
	- 형태적 변이들은 급격히 발전함

- 형태학적 관점에서 대진화의 경향
	- 생명체 크기의 증가 경향
	- 형태학적 복잡성의 증가 경향
		- 자연선택이 형질을 단순화시키기도 함

# 생물다양성
- 적응방산과 멸종의 결과로써 시간에 따라 변화함
- 적응방산
	- 급진적 종분화가 다른 서식지를 차지하거나 다른 먹이를 먹는 근연종들의 무리를 형성함
	- 하나의 조상종이 새로운 적응대로 이주한 후 나타남
		- 적응대: 일반적인 생활양식
	- 하나의 적응 지대는 한 성공적인 군의 소멸 후 열리기도 함
		- 공룡류 감소 후 포유동물의 폭발적인 적응방산
- 멸종
	- 유형
		- 바탕멸종
		- 대멸종
	- 배경멸종률: 종에게 낮은 비율로 기대되는 표준 멸종 속도
		- 환경의 변화에 적응하지 못해 생존 및 번식하지 못하는 종은 사라짐
		- 현재까지 살았던 99.9% 이상의 종들이 멸종
		- 1억 년당 50% 이상의 종이 멸종
		  -> 종의 지속적인 치환 과정으로 특징 지어지는 생명의 역사
	- 대멸종: 멸종률이 배경멸종률을 크게 초과했던 시기
		- 최소 5번의 시기
		- 페름기 멸종이 가장 심각
			- 고생대 -> 중생대
		- 백악기 멸종 기간 동안 공룡류 포함 지구상 50% 이상의 종이 멸종됨
			- 중생대 -> 신생대
		- 제 6대 대멸종 -> 인간의 환경파괴의 결과
		- 대멸종의 원인
			- 지구 판구조 활동과 그에 따른 기후변화
			- 운석 충돌 -> 백악기 대멸종
			  -> 의문점, 백악기 멸종은 수만 년에 걸쳐 일어남

- 생물다양성의 증가
	- 대멸종 -> 다른 생물들의 기회
	- 생태학적 상호작용의 진화
		- 속씨 식물로 인한 새로운 적응대
		  -> 백악기에 곤충의 급진적 분화
		  -> 곤충이 수분매개자로서 속씨식물의 적응방산 자극

- 진화발달생물학
	- 배 발생과정을 조절하는 유전자
	  -> 체형과 형태에 변화
	  -> 유전자의 진화적 변화 연구
	- 호미오 유전자: 생명체의 형성에 기여하는 유전자들의 발현을 조절하는 전사인자를 암호화하는 유전자
	- 유전체 염기서열의 비교
	  -> 현존하는 모든 동물 종들은 그들의 발달을 조절하는 수백 개의 호미오 유전자군을 공유
	  -> 이들 유전자군이 하나의 공통조상으로부터 유전됨
	  -> 호미오 유전자의 발현 변화가 형태에 큰 영향을 줄 수 있음

# 생물의 분류와 계통
- 계통분류학
	- 목표
		- 진화사의 재건
		- 종의 분류
	- 진화사의 재건
		- 계통 -> 계통수로 도해됨
		- 새로운 자료를 모아 고쳐나감
	- 종의 분류
		- 분류학: 종을 동정하고 이름을 붙이고 분류적 위치에 배열
		- 분기진화의 유형에 따라 분류
	- 생물 다양성 유형 연구
		- 소진화, 종분화, 적응방산, 멸종에 대한 이해 증진
	- 종 동정
		- 질병체(농업해충, 말라리아를 전파하는 모기)의 조절에 중요
		- 멸종위기종 식별
		- 야생동물을 효율적으로 관리
		- 선택적 교배와 유전공학을 위한 야생동식물 선택에 도움
	- 수집되고 체계화된 자료
		- 생물학자들의 연구를 위한 생명체 선택에 도움
	- 계통수 연구
		- 생물학자들의 진화적 과정 분석을 위한 비교방법의 요소

- 린네
	- 이명법: 생물에 이름을 부여하고 분류하는 기본 체계
		- 1. 속: 형태적으로 유사한 종들 식별
			- 첫 자는 항상 대문자
		- 2. 종소명: 종의 이름
			- 첫 자는 항상 소문자
		- 전체 종명은 이탤릭체
	- 분류계급: 생명체를 포함된 단위로 배열
		- 영역 > 계 > 문 > 강 > 목 > 과 > 속 > 종

- 린네: 외부 형태형질에 초점을 맞춤
		- 조류: 깃털, 2개 날개, 2개 발, 딱딱한 부리를 갖고 알을 낳는 동물의 강
- 오늘날 계통학자: 핵산과 단밸질의 분자서열의 비교분석 결과를 이용

- 형태형질
	- 살아 있는 표본에서 측정
	- 현존하는 종과 화석의 비교
	- 유용한 형태형질은 생물군에 따라 다름

- 행동형질
	- 외부형태가 비슷한 종을 구분
	- ex. 교미기에 특징적인 울음소리
		- 수정 전 생식적 격리기구
	- ex. 종 간의 염색체 수가 다름
		- 수정 후 격리기구 중 하나

- 분자계통학
	- 생명체 형질에 비해 풍부한 자료 제공
	- DNA 서열 자료 이용
		- PCR(중합효소연쇄반응) 이용
	- 우즈
		- rRNA 염기서열에 기초한 계통수
		- 살아 있는 생명체를 3개의 주요 영역으로 나눔
			- 세균영역, 고세균영역, 진행생물영역
